# Auditoria-Automatizada-de-Contratos-de-T.I.
Automação em Python focada na extração de dados críticos em arquivos PDF. Utiliza expressões regulares (Regex) para ler em lote, identificar e organizar automaticamente CNPJs, valores mensais e métricas de SLA de contratos de serviços tecnológicos e licenças de software, gerando planilhas consolidadas para controle de fornecedores.
